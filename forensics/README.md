Using Grep 

syntax: egrep 'whatisbeingsearchedfor' filename


-[abd] - included within the brakets

[^abd] - not included

{n} - preceding character matches exactly ntimes

++ - 1 or more times

? - 0 or 1 times only

. - a single character

** - 0 or more times

[c - g] - range

^ - matches beginning

$ - matches end
